# Bikers Peak 🚴‍♂️🏔️

Welcome to the official repository for **Bikers Peak**, hosted via GitHub Pages. 

This repository contains the source code, assets, and configuration files for the website.

---

## 🌐 Live Website
You can view the live site here: **[bikerspeak.github.com](https://bikerspeak.github.com)** (or your configured custom domain).

---

## 📂 Project Structure

Here is a quick overview of the essential files in this repository:

* **`index.html`** – The main landing page and entry point of the website.
* **`CNAME`** – Configures the custom domain pointing to this GitHub Pages site.
* **`robots.txt` & `sitemap.xml`** – Optimized for search engine indexing (SEO).
* **`favicon.svg`** – The browser tab icon.
* **`assets/`** – Contains images like `Bikers-Peak.png` and `bikers-peak.webp`.
